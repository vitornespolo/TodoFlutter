 ToDo é uma aplicação simples para guardar as suas tarefas diárias para não perder nada.

# Como executar o projeto
Pré-requisitos: SDK Flutter

```bash
# clonar repositório
git clone https://github.com/Joao3run/todo
# instalar dependências
flutter pub get
# executar o projeto
flutter run
```
